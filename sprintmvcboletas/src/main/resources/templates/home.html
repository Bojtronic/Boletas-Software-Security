Sistema de gestión de boletas

Con este sistema, remplaza por completo la necesidad de utilizar papeles para la gestión de las boletas de servicio. Permite crear boletas, asignar técnico y almacenar firma digital.
También la gestión de clientes y asignación de ellos a las boletas.
Es capaz de generar diversos reportes como: Servicios realizados, Usuarios recurrentes, productos utilizados, rendimiento de técnicos y tiempo de respuesta.
El sistema también maneja roles para los usuarios para asi poder asignar permisos dependiendo de su rol.
